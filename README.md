**Japanese Grammar atomic** — это интерактивный тренажер грамматики уровня JLPT N5, построенный на принципах микрообучения и глубокого визуального погружения. Мы не просто учим правила, мы проживаем их в контексте реального путешествия по Японии.

---

## ✨ Ключевые особенности

### 🏞️ Визуальное погружение
Каждый урок — это остановка в знаковом месте Японии (Сибуя, Асакуса, Киото). Высококачественные аниме-иллюстрации и динамический параллакс-фон создают атмосферу живого путешествия.

### 🎭 Обучение через диалог
Забудьте о сухих лекциях. Грамматику объясняют живые персонажи:
- **Маме-сиба**: Наш верный пушистый маскот.
- **Аой**: Коренная японка и ваш гид по культуре.
- **Кай**: Иностранный студент, который учится вместе с вами.

### 🧩 Игровая практика
- **Конструктор предложений**: Собирайте фразы из блоков, чувствуя структуру языка.
- **Мини-сцены**: Заполняйте пропуски в реальных диалогах.
- **Скрытые подсказки**: Переводы спрятаны под спойлерами, чтобы вы могли сначала проверить себя.

## 🛠️ Технологический стек

- **Core**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Premium animations & Glassmorphism)
- **Data**: Atomic lesson structures with custom metadata for storytelling.

## 🚀 Быстрый старт

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Откройте в браузере: `http://localhost:5173`

---

## 📚 Архитектура курса

Курс разбит на 20 атомарных уроков, покрывающих основные темы N5:
- Связки (です, ではありません)
- Частицы (は, の, も, を, に, へ, と)
- Прилагательные (い-adj, な-adj)
- Глаголы (ます-форма, прошедшее время)
- Счетные суффиксы и многое другое.

## 🎨 Дизайн-система

Приложение использует кастомную дизайн-систему, вдохновленную современными японскими интерфейсами:
